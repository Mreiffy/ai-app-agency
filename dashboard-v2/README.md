# AI App Agency - Command Center

A premium, interactive dashboard for managing your autonomous AI agent team.

## Features

- **Real-time Agent Monitoring** - See what each agent is doing with live status updates
- **Interactive Chat** - Talk to any agent directly
- **Activity Feed** - Live stream of all agent actions
- **System Metrics** - Cost tracking, API usage, build stats
- **Quick Actions** - One-click commands to any agent

## Tech Stack

- React + TypeScript
- Tailwind CSS
- Vite
- Lucide Icons
- date-fns

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

```bash
vercel --prod
```
