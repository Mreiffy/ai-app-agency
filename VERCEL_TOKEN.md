# Vercel Deployment Token Setup

## Get Your Vercel Token

1. Go to https://vercel.com/account/tokens
2. Click **Create Token**
3. Name it: `AI Agency`
4. Click **Create Token**
5. **COPY the token** (starts with `vercel_...`)
6. **Paste it here**

Then I'll deploy automatically.

## Or Deploy Manually

1. Go to https://vercel.com/new
2. **Import Git Repository**
3. Select: `Mreiffy/ai-app-agency`
4. **Configure:**
   - Framework: **Other**
   - Root Directory: `dashboard`
   - Build Command: (leave empty)
   - Output Directory: `./`
5. Click **Deploy**

Your dashboard will be live at `https://ai-app-agency.vercel.app`
